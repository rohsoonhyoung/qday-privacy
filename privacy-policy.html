개인정보 처리방침

최종 수정일: 2025년 11월 12일

시행일: 2025년 11월 12일

개발자: Unibox Labs

Unibox Labs(이하 "회사")는 Q-Day 애플리케이션(이하 "앱")을 무료 앱으로 제공합니다. 이 서비스는 회사에 의해 무료로 제공되며 있는 그대로 사용하도록 의도되었습니다.

이 페이지는 누구든지 앱을 사용하기로 결정한 경우 개인정보의 수집, 사용 및 공개에 관한 개발자의 정책을 알리는 데 사용됩니다.

1. 수집하는 정보

1.1. 자동으로 수집되는 정보

앱은 다음의 정보를 자동으로 수집할 수 있습니다:

  • 익명 사용자 식별자: Firebase Anonymous Authentication을 통해 생성된 익명 ID
  • 기기 정보: 기기 모델, 운영 체제 버전, 앱 버전
  • 사용 통계: Firebase Analytics를 통한 앱 사용 패턴 (화면 조회, 이벤트 등)
  • 감정 투표 데이터: 선택한 감정 (익명으로 저장됨)
  • 투표 날짜 및 시간: 감정 투표를 한 날짜

1.2. 로컬에만 저장되는 정보

다음 정보는 사용자 기기에만 저장되며 서버로 전송되지 않습니다:

  • 질문 답변 내용: 사용자가 작성한 모든 답변
  • 알림 설정: 알림 시간 및 활성화 여부
  • 테마 설정: 라이트/다크 모드 선택

2. 정보 사용 목적

수집된 정보는 다음의 목적으로만 사용됩니다:

  • 실시간 감정 통계 제공: 다른 사용자들의 감정 분포를 익명으로 표시
  • 중복 투표 방지: 하루에 한 번만 투표하도록 제한
  • 앱 개선: 사용 패턴 분석을 통한 기능 개선
  • 광고 표시: AdMob을 통한 맞춤 광고 표시

3. 정보 공유 및 공개

개발자는 다음의 경우를 제외하고는 사용자의 정보를 제3자와 공유하거나 판매하지 않습니다:

3.1. 서비스 제공자

앱은 다음의 제3자 서비스를 사용합니다:

Google Firebase

  • Firebase Authentication (익명 인증)
  • Firebase Firestore (감정 통계 저장)
  • Firebase Analytics (사용 통계)
  • Firebase 개인정보 처리방침

Google AdMob

  • 광고 표시 및 맞춤 광고
  • AdMob 개인정보 처리방침

3.2. 법적 요구사항

법률에 의해 요구되거나 정부 기관의 합법적인 요청이 있는 경우 정보를 공개할 수 있습니다.

4. 데이터 보안

개발자는 사용자 데이터의 보안을 중요하게 생각합니다. 하지만 인터넷을 통한 전송 방법이나 전자 저장 방법이 100% 안전하다고 보장할 수는 없습니다.

보안 조치:
  • Firebase의 보안 규칙 적용
  • HTTPS를 통한 암호화된 데이터 전송
  • 질문 답변은 로컬 기기에만 저장 (전송되지 않음)

5. 데이터 보관 및 삭제

5.1. 보관 기간

  • 감정 투표 데이터: Firestore에 영구 보관 (익명)
  • 질문 답변: 사용자가 앱을 삭제할 때까지 기기에 보관

5.2. 데이터 삭제

사용자는 다음의 방법으로 데이터를 삭제할 수 있습니다:

  • 로컬 데이터 삭제: 앱 설정 → 데이터 초기화
  • 앱 삭제: 앱을 삭제하면 로컬 데이터가 모두 삭제됨
  • 서버 데이터 삭제: 익명 감정 투표 데이터는 통계 목적으로 보관되며, 개인을 식별할 수 없음

사용자가 모든 데이터를 삭제하고 싶은 경우, 아래 이메일로 연락주시면 24시간 내에 처리해드립니다.

6. 사용자 권리

사용자는 다음의 권리를 가집니다:

  • 열람권: 수집된 정보를 확인할 권리
  • 정정권: 부정확한 정보를 수정할 권리
  • 삭제권: 정보 삭제를 요청할 권리
  • 처리 정지 요청권: 정보 처리 정지를 요청할 권리

7. 쿠키 및 추적 기술

앱은 다음의 기술을 사용합니다:

  • Firebase Analytics: 앱 사용 통계 수집
  • AdMob: 광고 표시 및 추적

사용자는 기기 설정에서 광고 맞춤 설정을 변경할 수 있습니다:

  • Android: 설정 → Google → 광고 → 광고 맞춤설정 해제

8. 개인정보 처리방침 변경

개발자는 때때로 이 개인정보 처리방침을 업데이트할 수 있습니다. 변경사항은 이 페이지에 게시되며, 중요한 변경사항이 있는 경우 앱 내에서 알립니다.

사용자는 정기적으로 이 페이지를 확인하여 변경사항을 파악하는 것이 좋습니다.

9. 동의

앱을 사용함으로써 사용자는 이 개인정보 처리방침에 동의하는 것으로 간주됩니다.
